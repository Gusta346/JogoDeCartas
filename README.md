# Jogo de Cartas - C#

Este é um projeto simples em C# (.NET 8) criado para representar a estrutura básica de um jogo de cartas. O objetivo principal do projeto é demonstrar o uso de Programação Orientada a Objetos, com a criação de classes, enumerações, propriedades e métodos para simular um baralho, jogadores, mãos e rodadas de jogo.
